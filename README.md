# **网络安全开源图书三部曲**

**【写在前面】**

@ t0data

**感谢您的阅读！值2025年国家网络安全宣传周之际，谨以此系列图书献给那些有安全梦想的人！**

去年有一段时间在微信群组织大家一起安全打卡，讨论工作日常困惑和经验总结。于是乎，在这个过程中，沉淀了很多电子资料。

近两年，AI工作内容已经越来越多，纯安全工作越来越少，**与其珍藏这些知识，不如贡献给开源社区，帮助更多的安全人！**

所以，这次从2015年开始陆陆续续地做的开源图书中，挑选与网络安全相关的3本图书，整理成一个系列，称之为**网络安全开源图书三部曲**。

它们若能对您在安全之路的成长和发展有所助益，那便是我最大的心愿。

（若你是初学者，这几本书，建议你倒着来读）。

当然，由于个人水平有限，书中**错误，不足之处在所难免**。

如果你在阅读过程中发现问题，请留下批注或者去读者交流星球发言。
